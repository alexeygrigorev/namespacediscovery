\subsection{Parameter Tuning} \label{sec:param-tuning}

We have the following parameters

Way to incorporate definition information (3 ways):
no identifier, soft association, hard association 

Weighting schemes: TF-IDF, TF, log TF

Clustering algorithm

DBSCAN, SNN Clustering: params: min\_pts, epsilon 
K-Means, Bisecting K-Means: params: $k$ 


Distance and similarity measures used
Euclidean distance, cosine similarity, jaccard similarity, SNN Similarity

Ways to reduce dimensionality 
SVD $D \approx D_k = U \Sigma_k V$, param: $k$ what's the rank of the approximation matrix 
NNMF $D \approx D_k = U_k V_k^T$ param: $k$ number of columns in $U$ and $V$ - also the rank of $D_k$

The approach for finding the best parameter set is a grid search: different combination
are tries and the best result is kept.
 
Agglomerative: Wald linkage: takes forever never finished
